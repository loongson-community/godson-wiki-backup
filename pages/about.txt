===== 基础设施 (Infrasturctures) =====
本网站提供一系列基础设施，目前基础设施由[[user:flygoat|Jiaxun Yang]]维护，如果遇到任何问题，你可以直接联系他。

==== Wiki ====
=== wiki.godson.ac.cn ===
使用Dokuwiki搭建，主要用于分享信息，任何注册用户都可以编辑。

Built with DokuWiki, all registered users can edit.

==== FTP ====
=== ftp.godson.ac.cn ===
用于存储部分软件及文档，任何人都可以下载，但上传服务仅向授权用户开放，如果有需要申请帐号请联系维护者[[user:flygoat|Jiaxun Yang]]。

请注意服务器资源有限，请尽量不要上传有固定外部链接(例如来自GitHub)的资源。

也请不要上传可能涉及版权风险的文件。

For sharing files, everyone can download but only trusted users can upload. Please contact the maintainer [[user:flygoat]] if you want to be a trusted user.

To save server resources, please don't upload files with fixed external links (e.g. Files from GitHub).

Also don't upload files with copyright issues.
